User Stories

create and access account

As a student, I want to create an account so that I can join the platform.
As a user, I want to log in and log out securely so that I can access my personal account safely and protect my information.
profile creation
As a student, I want to create a profile with my details so that other students can know who I am and what skills I offer or want to learn.
listing and browsing the skills
As a student, I want to list skills I can teach so that other students can discover what I can offer.
As a student, I want to list skills I want to learn so that I can find other students who may be able to help me.
As a user, I want to browse skills posted by others so that I can explore available learning and teaching opportunities.
As a user, I want to search or filter skills so that I can quickly find relevant posts based on my interests.
As a user, I want to view detailed skill information so that I can decide whether a skill post matches my needs.
sending the request and approval
As a student, I want to send a skill exchange request so that I can express interest in learning from or exchanging skills with another student.
As a student, I want to receive requests so that I can see who is interested in my skill posts.
As a student, I want to accept or reject requests so that I can control which exchanges I want to participate in.

CRUD on the post

As a user, I want to edit or delete my posts so that I can keep my skill information accurate and up to date.
